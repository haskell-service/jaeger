jaeger-opentracing-tags
=======================
Utilities to construct [jaeger-types] `Tag`s and `Log`s according to the
[OpenTracing] [Semantic Conventions].

[jaeger-types]: https://github.com/haskell-service/jaeger/tree/master/jaeger-types/
[OpenTracing]: http://opentracing.io/
[Semantic Conventions]: https://github.com/opentracing/specification/blob/ea2b29f46500827083e42b7a0d65bd422ccc28c1/semantic_conventions.md
